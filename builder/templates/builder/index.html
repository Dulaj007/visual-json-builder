{% load static %} {% comment %} index.html — JSON Builder Single-Page Template ================================================ This is the only Django template in the project. Django renders it once on the initial GET request; all subsequent interaction is handled client-side by app.js without any page navigation. Template structure (top to bottom): head — meta, title, external stylesheet, inline toast CSS header — sticky top bar: logo + Preview / Clear / Export buttons main — two panels rendered inside .app-main Schema panel — schema field list + column headers + type legend Items panel — items container (filled by app.js) Preview modal — hidden overlay for the JSON preview dialog Toast element — slide-up notification (shown/hidden by app.js) script — loads app.js after the DOM is ready Django template tags used: load static — enables the static tag for resolving asset URLs static 'path' — resolves a static asset to its full URL at render time & in panel title — HTML entity for the & character All interactive behaviour (rendering items, exporting JSON, etc.) lives in app.js. The HTML only provides static markup and the anchor IDs that JavaScript queries: #schema-fields, #items-container, #preview-modal, #toast, #schema-count, #items-count, #preview-code. {% endcomment %} JSON Builder
